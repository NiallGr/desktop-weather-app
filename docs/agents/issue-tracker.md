# Issue tracker

This project's Feature and User Story tracker is **Azure DevOps**.

- **Organisation:** Enate (`https://dev.azure.com/Enate`)
- **Project:** Nialls Factory DevTest
- **Work item types:** Epic (PRD), Feature (one per Roadmap Feature), User Story (per `enate-to-stories`).
- **Ready-for-agent signal:** the `ready-for-agent` tag.

Skills that publish work items (`/to-prd`, `/publish-feature`, `/enate-to-stories`, `/to-issues`)
must target Azure DevOps via the ADO MCP — **not** GitHub Issues. The GitHub repo holds code and
the planning docs; it is not the work-item board.

Footer convention for published artefacts: `> ADO Work Item: <url>`.
